# Password Generator API

Полнофункциональное Express-приложение для генерации безопасных паролей с веб-интерфейсом и REST API.

## Возможности

- ✓ Веб-интерфейс для генерации паролей
- ✓ REST API с поддержкой GET и POST запросов
- ✓ Оценка уровня сложности пароля
- ✓ Настройка наборов символов (прописные, строчные, цифры, символы)
- ✓ Длина пароля от 6 до 32 символов
- ✓ Логирование всех запросов
- ✓ Обработка ошибок на бекенде
