# Генератор Паролей
## Описание проекта
Полнофункциональное веб-приложение для генерации безопасных случайных паролей. Приложение позволяет пользователю создавать пароли с настраиваемыми параметрами, включая размеры и символы (прописные буквы, строчные буквы, цифры, специальные символы).
## Основные возможности
**Генерация паролей** — случайных паролей с заданными параметрами
**Выбор паролей** — пароли от 6 до 32 символов
**Выбор состава** — прописные (AZ), строчные (az), цифры (0-9), специальные символы (!@$#%&)
**Оценка надёжности** — визуальная шкала и текстовое описание уровня безопасности
**Копирование в буфер обмена** — один клик для ввода
**Проверка входных данных** — проверка корректности минимума и набора параметров

## Особенности реализации
**Интуитивный интерфейс** — чистый и минималистичный дизайн
**Архитектура** — Файловое HTML5-приложение — весь код в одном файле (HTML + CSS + JavaScript)
**Ванильный JavaScript** — без зависимости от библиотек
**Адаптивный дизайн** — работает на всех устройствах
**Асинхронная обработка** — не блокирует интерфейс

## Технические решения
1. **Генерация случайных чисел** — Использует `Math.random()` для выбора символов.
2. **Перемешивание пароля для случайного определения**.
3. **Оценка надёжности** — Использует эвристический алгоритм оценки. 
4. **Валидация** — Проверка длины (6-32). Проверка ввода специальных групп. Вывод ошибок на русском языке.
4. **Дизайн-система** — Переменные CSS для цветов и отступов. Поддержка светлого режима (основной) - Современные стили (border-radius, box-shadow, переходы)
